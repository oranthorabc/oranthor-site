About Oranthor

Oranthor is a peer-to-peer financial network where every mobile device becomes a node. It enables lightning-fast, anonymous transactions without central servers.

Powered by $ORR and fiat-pegged ORX tokens, Oranthor removes all friction from modern payments.

Our vision is to create a truly decentralized financial ecosystem that works seamlessly across borders, devices, and currencies.

How It Works

Peer-to-Peer Protocol

Uses quorum validation (k/n) to verify transactions without a central authority, ensuring security and speed.

One-Tap Wallets

Wallets created in 1 tap, no KYC required. Instant access to the Oranthor ecosystem.

Encrypted Balances

Using AES-256 + Shamir Secret Sharing to ensure your funds remain private and secure at all times.

Dynamic Node Expansion

No global blockchain needed. Shard storage using Arweave + Bundlr for efficient data management.

Frequently Asked Questions

How is ORX different from USDT?

Unlike USDT which is backed by Tether's reserves, ORX tokens are fully collateralized by ORR at a 150% ratio. This means each ORX token is backed by $1.50 worth of ORR, providing greater stability and transparency. Additionally, ORX transactions have zero gas fees and settle in under 1 second.

Is KYC required to use Oranthor?

No, Oranthor is designed to be permissionless and privacy-preserving. You can create a wallet and begin using the network without KYC. However, certain fiat on/off ramps may require KYC due to local regulations, but the core protocol remains KYC-free.

How does the quorum validation work?

Oranthor uses a k/n quorum system where a transaction is validated when k out of n randomly selected nodes reach consensus. This provides security without requiring every node to validate every transaction, enabling our sub-second finality. The quorum size scales dynamically based on transaction value and network conditions.

What happens if I lose my device?

Oranthor uses a recovery system based on Shamir's Secret Sharing. During wallet setup, you can designate trusted recovery contacts or store recovery shards. With these shards, you can restore your wallet on a new device. We recommend setting up multiple recovery options for maximum security.

How can I earn with Oranthor?

There are multiple ways to earn with Oranthor:

  • Stake ORR to earn protocol fees
  • Run a validator node to earn validation rewards
  • Provide liquidity for ORX minting
  • Participate in governance to earn proposal rewards
